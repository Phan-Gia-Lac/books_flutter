Mobile Testing App - Flutter (Android)
Dự án này được xây dựng để thực hành quy trình kiểm thử phần mềm trên thiết bị di động, tập trung vào việc xác minh các yêu cầu đặc tả (SRS) thông qua Framework Flutter.

🛠 Yêu cầu hệ thống (Prerequisites)
Trước khi bắt đầu, hãy đảm bảo máy tính của bạn đã cài đặt:

Flutter SDK: Phiên bản 3.41.6 (Stable channel).

Android SDK: API 34 hoặc 35.

Cấu hình Toolchain: Chạy flutter doctor --android-licenses để chấp nhận tất cả các điều khoản.
